# Vision

## The One Line
A cross-platform study companion that tracks whether you're actually focused, tests if you actually learned, and builds a personal picture of how you study best — without ever interrupting your flow.

## The Problem
Students lie to themselves about how much they study. They sit for 3 hours and call it a 3 hour study session. In reality they:
- Picked up their phone 15 times
- Drifted into YouTube
- Stared at the wall daydreaming
- Were genuinely focused for maybe 45 minutes

No existing tool catches this honestly. Every other app measures time. This measures attention.

## The Target User
- Primary: High school and college students
- Secondary: Anyone doing deep focused knowledge work

## The Core Design Philosophy
```
Before session  → set intentions, declare study mode
During session  → completely silent while focused
                → speaks ONLY when distraction detected
After session   → full honest breakdown
Over time       → personal study intelligence
```

## The Golden Rule
**Only interrupt the user when they are already interrupted.**

## The Single Promise
"The time you spend studying with this app is real study time. Not fake study time."
